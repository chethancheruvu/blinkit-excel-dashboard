# Blinkit-excel-dashboard
Interactive Excel dashboard for BlinkIT grocery sales analysis with KPIs, Pivot Tables, Charts

# Project Overview
This project analyzes BlinkIT grocery sales data and presents key business insights through an interactive dashboard. The dashboard helps stakeholders understand sales performance, outlet trends, product categories, customer ratings, and other important KPIs.

# Process
 -BlinkIT Grocery Sales Dashboard built using Microsoft Excel to analyze sales performance and customer trends.

 -Provides insights into outlet performance, product categories, sales distribution, and customer ratings.
 -Designed to support data-driven decision-making through clear and interactive visualizations.
